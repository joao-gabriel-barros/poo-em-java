# Prática de Programação Orientada a Objetos com Java ☕

Repositório dedicado ao estudo e aplicação dos pilares de **POO** (Programação Orientada a Objetos). O objetivo é consolidar o aprendizado através de exercícios práticos e implementação de sistemas simples.

## 🎯 Conceitos Praticados
* **Abstração:** Criação de classes representando entidades do mundo real.
* **Encapsulamento:** Uso de modificadores de acesso e métodos acessores.
* **Herança:** Estruturação de hierarquia entre classes para reaproveitamento de código.
* **Polimorfismo:** Implementação de métodos que se comportam de diferentes formas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Controle de Versão:** Git & GitHub
